## 开发计划

### 一期

打通基础餐品付费链路，记录消费订单；配套简单的后台系统，展示基础数据报表

### 二期

实现餐品选择流程的拓展，优化视觉效果，后台提供更全面的报表，提供快捷配置能力支持

### 三期 -- release demo

添加头部运营banner，配套头图配置后台；提供更细粒化的曝光数据

### 四期

从响应速度，首屏时间，图片加载时间等纬度优化用户体验

### 五期

进行更多功能拓展和产品体验
